ISLAND FORGE - MAP LOCATIONS REFERENCE
=====================================

Purpose
-------
This file defines the first launch map set and reserves later island concepts.
Use this for future map, foraging, cosmetics, market, and encounter design.

Launch map: first 5 locations
-----------------------------

1. Bermuda Abyss
- Inspiration: Bermuda Triangle, vanishing ships, compass failure, time skips, ocean horror.
- Gameplay role: starter mystery island and storm-material zone.
- Primary resources:
  - storm_iron_ore
  - black_salt
  - rift_whisper_algae
  - ghostwood_resin
  - coral_shards
- Cosmetic themes:
  - salt-lantern charms
  - hull-stitched cloaks
  - moon shroud masks
- Talent links:
  - Darkness/Shadows
  - Storm/Tempest
  - Luck/Fate
  - Navigation/Cartography
- Main activities:
  - first foraging routes
  - shipwreck salvage
  - cursed compass events
  - storm route tutorials

First implementation recommendation
-----------------------------------
- Keep `Bermuda Abyss` as default starting island.
- Add all five launch locations to map/forage data before expanding.
- Make each island own:
  - material pool
  - cosmetic pool
  - risk profile
  - preferred talent domains
  - one signature event hook
- Do not build combat or full quest systems yet. Start with travel, forage, discovery, and market supply.
